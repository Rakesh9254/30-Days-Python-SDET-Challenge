# Practice Questions: Input & Output

## Easy
1. **Greeting Script:** Write a script that asks the user for their first name and last name separately, and prints a greeting message like "Hello, [First] [Last]!".
2. **Simple Calculator:** Ask the user to input two numbers. Print the sum, difference, product, and quotient of those numbers.

## Medium
3. **Receipt Generator:** Ask the user for the name of an item, the price, and the quantity. Calculate the total cost. Print a formatted receipt that aligns the item name, price, quantity, and total cost neatly.
4. **Environment Configurator:** Write a script that takes a base URL (e.g., `google.com`) and an environment prefix (e.g., `qa`, `dev`). Use string formatting to print the full URL: `https://qa.google.com`.

## Hard
5. **QA Test Summary Report:** 
   - Ask the user to input the number of Total Tests, Passed Tests, and Failed Tests.
   - Calculate the Pass Percentage and Fail Percentage.
   - Print a beautifully formatted report using `f-strings` and multi-line strings.
   - Ensure percentages have exactly 2 decimal places.
   - *Bonus:* Try to right-align the numbers in your output.
