# Practice Questions: Day 01

## Easy
1. Install Python on your machine and verify the version via the terminal. Take a screenshot.
2. Install VS Code and the Python extension.
3. Write a Python script that prints your name and your current job title. Run it from the terminal.

## Medium
1. Create a script called `sys_info.py`. Import the `sys` module and print out the current Python version programmatically using `sys.version`.
2. Write a script that deliberately contains a syntax error (e.g., missing a closing parenthesis on a print statement). Run it from the terminal and observe what a Python Error Traceback looks like. 

## Hard
1. **(QA Focused):** Research how to set up a `.gitignore` file. Create a new git repository in your challenge folder, add a `.gitignore` specifically for Python projects (ignoring `__pycache__`, `.vscode`, etc.), and make your first commit.

## Mini Challenge
Write a Python script `sdet_goals.py` that prints 3 lines:
- Line 1: Your current role.
- Line 2: Your target role (Python SDET).
- Line 3: One tool you want to learn (e.g., Pytest, Playwright).
Run this script both from the IDE and the Terminal.

## Mini Project
Initialize your own GitHub repository for this 30-Day challenge and push your `Day 01` folder to it!
